Xcode Alternative Icons

Posted on 2019, Sep 23 by Matthias Buchetics

Xcode beta season is over but it’s still the time of the year where iOS developers usually have multiple Xcode version installed. Distinguishing one from another isn’t easy if you have multiple identical icons in the dock and, like me, getting reminded by Xcode that your project’s Swift version isn’t compatible.

Alternative icons to the rescue! It’s actually quite easy to change any app’s icon in macOS. After opening the info panel of Xcode.app you can select the current icon by clicking on it.

You can then copy (Command + C) the icon, edit it in your favorite graphics editor and paste Command + V) it back into the info panel to set the new icon and overwrite the old one (a resolution of 1024x1024 pixels is recommended).
