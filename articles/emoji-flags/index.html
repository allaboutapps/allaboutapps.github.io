Emoji Flags

Posted on 2019, Jan 23 by Gunter Hager

Sometimes there’s the need to fly a flag in your project. You probably know the ISO code, more specifically the ISO 3166-1 alpha-2 code, a.k.a two-letter code. For Austria this would be AT. Now when thinking about converting these codes into emoji flags, a lookup table comes to mind. But instead of jumping straight into implementing the table it’s time to take a closer look at the Unicode Standard. The clever people that added the flag emojis to the Unicode character set invented a system of specifying the flags that makes an algorithmic conversion without lookup tables quite easy.

Regional Indicator Symbols

Each flag emoji defined in the Unicode standard is created by concatenating two regional indicator symbol characters. func emoji(isoCode: String) -> String? {
  let code = isoCode.localizedUppercase
  // Check if ISO code is a legal region code
  guard Locale.isoRegionCodes.contains(code) else { return nil }
  // Calculate scalar value offset
  let offset = UnicodeScalar("ðŸ‡¦").value - UnicodeScalar("A").value
  // Convert ISO code characters into regional indicator symbols
  let characters = code.unicodeScalars.compactMap { UnicodeScalar($0.value + offset) }
  // Check if 2 characters were converted
  guard characters.count == 2 else { return nil }
  // Combine characters into string
  let result = String(String.UnicodeScalarView(characters))
  // Check if result is a valid flag emoji (otherwise count would be 2)
  guard result.count == 1 else { return nil }

  return result
}

I’ve added a couple of checks to the function to make sure that only valid flag emojis are returned.
